# Usage

1. Install .NET Core
2. Install NodeJS
3. Run the `RunDemo.ps1` script from PowerShell.
4. Open http://localhost:9000 in your browser. 

To stop the demo, run `kill-demo` or close all five console windows that the demo opened.
