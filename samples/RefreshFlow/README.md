# Usage

1. Install .NET Core
2. Install NodeJS
3. Run the `RunDemo.ps1` script from PowerShell.
4. Wait approx 5 minutes for packages to restore and apps to run

To stop the demo, run `kill-demo` or close all two console windows that the demo opened.
